Lesson 1 Notes

The Basics of the Web and HTML

What is a web page?

A web page is a document written in HTML (Hypertext Markup Language). Web browsers read these documents, interprets and then displays them.

How coding works

Programmers write text in a language (HTML) that computers can understand.

Computers are stupid

Programmers need to write with correct syntax so computers can understand us.

Programmers can't remember everything

Programmers can use other resources and tools to help with coding.

Some Basic HTML vocabulary:

• Tag: command or instruction.

• Element: everything within a set of an opening & closing tag. There are 2 types of elements: inline and block. Inline is simply text while a block element has an invisible box around the content. For example, "br" (break) is inline and "p" (paragraph) is a block element.

• Attribute: property of a element. (For example, a link to a web page.)

Lesson 2 Notes

Creating a structured document with HTML

HTML

The language used to create webpages (the nuts and bolts).

CSS (Cascading Style Sheets)

Helps determine how a webpage looks (colors, fonts, etc.).

DOM (Document Object Model)

How everything in a webpage is organized in a tree structure.

Boxes, boxes boxes!

All the elements on a webpage are organzied and broken down into boxes. Think of headlines, text, photos, etc. with an invisible box around each element. These "boxes" can be rerranged and moved around on the page.

Lesson 3 Notes

Adding CSS Style to HTML Structure

Avoid repetition!!!

Avoiding repetition will make things way easier!

It will help avoid errors: If you don't avoid repetition, you'll have to do the same thing over and over again. This may mean copy and pasting certain style attributes into many HTML elements or rewriting the same code (sometimes with minor differences) many times. This can lead to errors if you decide to make a change to something.

CSS

CSS (Cascading Style Sheets) give programmers a way to control the style (the look) of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

CSS Selectors

Perhaps the biggest key to understanding CSS is understanding selectors. Selectors are what allows you to target specific HTML elements and apply style to them.

A “selector” is the instruction in a CSS rule set that tells the browser what elements to ‘select’ for styling.

The Box Model

The CSS box model is essentially a box that wraps around HTML elements, and it consists of: margins, borders, padding, and the actual content. The box model allows us to add a border around elements, and to define space between elements.

Remember, it's all about the boxes!
